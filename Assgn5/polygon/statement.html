Representing a Polygon

A polygon can be representing by listing its vertices in a clockwise or anti-clockwise manner. Implement a structure Point to store the coordinates of a vertex, and use a circularly linked list to store the vertices in a clockwise manner. We will then use these primitive data structures to define functions on the polygon.

Circularly Linked Lists

The circularly linked list is a slight modification of the conventional singly linked list in which the last element is linked to the first element.

Implement a circularly linked list with at least the following operations:

  • CLinkedList* clinked_list_new(): Create a new empty list.
  • void clinked_list_delete( CLinkedList* lst ): Deletes the list, frees memory.
  • int clinked_list_size( CLinkedList* lst ): Returns the number of elements in the stack
  • CLinkedList* clinked_list_insert( CLinkedList* lst, int i, Point val ): Inserts val at the 'i-th' index of the list.
  • CLinkedList* clinked_list_push_front( CLinkedList* lst, Point val ): Inserts val at the beginning of the list.
  • CLinkedList* clinked_list_push_back( CLinkedList* lst, Point val ): Inserts val at the end of the list.
  • CLinkedList* clinked_list_remove( CLinkedList* lst, int i ): Removes element at the 'i-th' index
  • CLinkedList* clinked_list_pop_front( CLinkedList* lst ): Removes the element at the beginning of the list.
  • CLinkedList* clinked_list_push_back( CLinkedList* lst ): Removes the element at the end of the list.
  • Point* clinked_list_get( CLinkedList* lst, int at ): Gets a pointer to element at the 'i-th' index
  • Point* clinked_list_front( CLinkedList* lst ): Gets the element at the beginning of the list.
  • Point* clinked_list_back( CLinkedList* lst ): Gets the element at the end of the list.
  • void clinked_list_print( CLinkedList* lst ): Prints the elements currently in the stack

Representing a Polygon

Implement a Polygon using the following functions:

  • Polygon* polygon_new(); Create a new polygon - with no vertices
  • void polygon_delete( Polygon* poly ); Deletes the polygon, freeing all vertices.
  • Polygon* polygon_add_vertex( Polygon* poly, Point p ); Add a vertex
  • Polygon* polygon_remove_vertex( Polygon* poly, Point p ); Remove a vertex
  • int polygon_size( Polygon* poly ); Returns the number of vertices in the polygon
  • void polygon_print( Polygon* poly ); Prints the vertices in the polygon
  • Point polygon_centroid( Polygon* poly ); Find the centroid of the polygon
  • double polygon_area( Polygon* poly ); Find the area of the polygon. You can do this by summing up the areas of each triangle formed between two vertices and the centroid.

Sample Input/Output

Input: N x1 y1 x2 y2 ...: The first number (N) will be the number of vertices, followed by 2N numbers which will represent the x and y coordinates of the N vertices. You can assume the inputted polygon is convex.
Output: area: The code will return the area of the polygon thus formed. Use the functions you've defined above to do this.
