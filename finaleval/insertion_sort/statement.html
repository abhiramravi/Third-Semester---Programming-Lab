Insertion Sort

Note: All code should be written within this template directory. Test cases have been provided in the folder tests, and can be run using make test.

Insertion sort is a very popular, simple iterative sorting algorithm. A brief description of the algorithm is as follows; search for the smallest element in the list and move it to the beginning of the list. Repeat the process with the rest of list. Implement the algorithm to sort integer arrays.

Sample Input/Output

Input: 5 3 4 7 3 -5
Output: -5 3 3 4 7
